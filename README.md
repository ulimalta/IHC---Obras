##Pré-requisitos
***************
Ter a última versão do xcode e do OS X Mavericks instalado no computador.

##Simulação
***************
1) Faça download do repositório.
 
2) Abra Obras.xcodeproj com o xcode.
 
3) Pressione "Run" para iniciar a simulação.
